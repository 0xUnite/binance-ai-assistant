# Binance AI Assistant

🤖 4 AI-powered tools for Binance ecosystem

## Projects

| Project | Description |
|---------|-------------|
| [trading-bot](./trading-bot/) | RSI-based automated trading bot |
| [portfolio-tracker](./portfolio-tracker/) | Real-time portfolio tracking |
| [signal-alerts](./signal-alerts/) | Price alert system with Telegram |
| [crypto-educator](./crypto-educator/) | AI-powered Q&A assistant |

## Quick Start

```bash
# Clone or download
cd binance-ai-assistant

# Setup any project
cd trading-bot
pip install -r requirements.txt
export BINANCE_API_KEY="your_key"
export BINANCE_SECRET_KEY="your_secret"
python main.py
```

## Requirements

- Binance API key (get from binance.com)
- Python 3.8+
- For crypto-educator: OpenAI API key
