# Skill: Token Audit

## 目标
对代币做基础风控检查，识别 honeypot、流动性与常见安全风险。

## 对应代码
- `utils/honeypot_detector.py`
- `web_ui/main.py` → `/api/honeypot/<token>`

## 适用场景
- 新币能不能碰？
- 这个地址是不是蜜罐？
- 上车前先做一个快速安全筛查

## 输入
- `token_address`
- 可选：链名称（BSC / Base / Solana 等）

## 输出
- 风险等级：低 / 中 / 高
- Honeypot 检查结果
- 流动性 / 交易限制 / 黑名单等风险提示
- 建议动作：继续观察 / 小仓试错 / 直接放弃

## 建议工作方式
- 先给一个一句话结论
- 再列出命中的风险项
- 最后说明“不构成完整审计”，避免误导

## 风险提示
- 这是快速筛查，不是完整合约审计
- 任何小币交易都应默认高风险
